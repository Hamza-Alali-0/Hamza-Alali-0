  # Suivi Doctorat

  A concise, professional README for the Suivi Doctorat workspace. This project contains multiple services and a frontend application used for managing doctoral follow-up processes.

  ## Overview

  This repository is a multi-module project that includes microservices (Spring Boot), a gateway, a configuration server, an Eureka server, and an Angular frontend. The workspace is intended for development and deployment of a doctoral tracking system.

  ## Features

  - Microservices for authentication, registration, notifications, and defense scheduling
  - Centralized configuration and service discovery
  - Angular frontend for user interactions
  - Uploads storage for user assets

  ## Technologies

  - Java, Spring Boot
  - Maven
  - Angular
  - Eureka (service discovery)
  - Spring Cloud Config

  ## Getting Started

  Prerequisites:

  - Java 11+ (or the version required by each module)
  - Maven 3.6+
  - Node.js + npm (for the frontend)

  Quick start (development):

  1. Build backend modules:

     mvn -T 1C clean install

  2. Start the services (each module in its folder) or use your IDE run configurations.

  3. Start frontend:

     cd frontend-app
     npm install
     npm start

  Adjust ports and configuration values in `config-repo` as needed.
